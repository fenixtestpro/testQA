

1) pwd-утилита, которая позволяет вывести в терминал путь к текущей папке.

2) mkdir-создаем папку.

   Пример: mkdir папка- В текущей директории будет создана папка с указанным именем.
   mkdir Загрузки/папка-1 Утилита создаст папку в подпапке загрузки.
   mkdir /home/имя_вашего_пользователя/папка
   mkdir {папка1,папка2,папка3} создаемсразу 3 папки. 
   mkdir -p ~/Загрузки/папка/{папка1,папка2,папка3} создаем в загрузках папку и в енй еще 3 папки.

3) cd-Позволяет перейти из текущего каталога в указанный.
   пример: cd	Возвращает вас в ваш домашний каталог
           cd ~	Также возвращает вас в ваш домашний к аталог
           cd /	Переносит в корневой каталог
           cd /root	Переносит в домашний каталог пользователя root. для доступа к этому каталогу вы должны быть суперпользователем
           cd /home	Переносит в каталог home, где обычно находятся каталоги всех пользователей, существующих в системе
           cd ..	Перемещает на один уровень вверх
           cd ../../dir3/dir2	Этот относительный путь сначала переместит вас на два уровня вверх, затем в каталог dir3, а затем в каталог dir2

4) ls- вывести на экран список файлов и каталогов текущего каталога.
   ls _la-покажет скрытые паки,выводит всю информацию о файле.
   ls -al /etc- что имеется в каталоге /etc/, находясь при этом в вашем домашнем каталоге.

5) vim имя файла.txt nanoимя файла.txt- откроет файл txt в указанном редакторе.

6) touch- создание файла.
   touch file.txt- создаем файл file.txt
   touch app/{css/main.css,js/main.js,index.html} — Создать сразу несколько файлов, никаких лишних пробелов

7) mv -Перемещение или переименование файлов и каталогов.
   dir/log.txt dir2 -перемещает файл lo.txt из дирректории dir в dir2
   mv file newfile - переименовывает file в newfile

8) cp-Копирование файлов и каталогов.
   cp log.txt dir2 копирование файла log.txt в директорию dir2
   cp log.txt/dir2 log1.txt/dir2 dir3 копирование 2ух файлов в dir3
9) find- поиск файлов по имени.
   find . -name linuxConsoleComand.txt

10) grep- поиск определенных строк в файле.
    grep файл linuxConsoleComand.txt -поиск слова файл в  linuxConsoleComand.txt

11) head- чтобы просмотреть начало файла.
    head -2 linuxConsoleComand.txt- просмотр первых 2ух строк файлаю.

12) tail -чтобы просмотреть начало файла.
    tail -2 linuxConsoleComand.txt- последние 2 строки файла.

13) less- less — она позволяет перематывать текст не только вперёд, но и назад
            
14) date- показывает дату и время.

15) curl- это загрузка файлов linux.
    curl https://losst.ru/kak-polzovatsya-curl

16) bash - запуск скрипта с командной сроки.
